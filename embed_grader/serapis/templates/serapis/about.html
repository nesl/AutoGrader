AutoGrader is a system developed by Networked and Embedded System Laboratory (NESL, also check out our lab website :). We've witnessed a lot of software automatic grading systems in programming competition domain (e.g., several online judges), in education domain (e.g., MOOCS), and most companies in software industry have developed mature testing pipeline to enhance code quality. So far we didn't see any system dedicated for automation of grading cyber physical systems. We're happy to announce our first version of AutoGrader system, and so excited that our system can assist you in EE202A/CS213A Embedded System course, instructed by our PI Prof. Mani Srivastava. We hope you find our AutoGrader system useful and enjoy the process of homework submissions. If you have any question or feedback you want to share with us, please email us at autograder@nesl.info.
